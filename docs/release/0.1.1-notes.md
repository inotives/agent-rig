# AgentRig v0.1.1

Patch release for the Phase 7 and Phase 8 MVP improvements.

## Highlights

- Added editable agent profiles and profile commands.
- Added built-in `researcher` and `writer` profiles.
- Added profile-declared shared and agent-local skill installs.
- Added `agent-rig doctor`, `agent-rig --version`, and improved help output.
- Added shared Markdown task files under `.agent-rig/_shared/tasks/`.
- Added `agent-rig tasks create`, list, JSON list, and show commands.
- Added warning-first task validation for metadata, dependencies, assignees, and acceptance criteria.

## Notes

- The package name remains `@inotives/agent-rig`.
- The CLI binary remains `agent-rig`.
- npm publish remains manual for this release because maintainer two-factor authentication is required.
