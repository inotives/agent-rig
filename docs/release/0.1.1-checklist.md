# AgentRig 0.1.1 Release Checklist

## Scope

- Patch release for Phase 7 CLI/profile polish.
- Patch release for Phase 8 Markdown task management.
- Manual npm publish with maintainer two-factor authentication.

## Pre-Release

- [ ] Confirm working tree is clean.
- [ ] Confirm package version is `0.1.1`.
- [ ] Confirm `CHANGELOG.md` has a `0.1.1` entry.
- [ ] Confirm `docs/release/0.1.1-notes.md` is ready.

## Validation

- [ ] Run `npm test`.
- [ ] Run `npm --cache /tmp/agent-rig-npm-cache pack --dry-run`.
- [ ] Run local tarball smoke test:

```bash
npm pack --pack-destination /tmp --cache /tmp/agent-rig-npm-cache
tmp="$(mktemp -d /tmp/agent-rig-0.1.1.XXXXXX)"
cd "$tmp"
npm init -y
npm --cache /tmp/agent-rig-npm-cache install /tmp/inotives-agent-rig-0.1.1.tgz
npx --cache /tmp/agent-rig-npm-cache agent-rig --version
AGENT_RIG_SKIP_SKILLS=1 npx --cache /tmp/agent-rig-npm-cache agent-rig init --yes
npx --cache /tmp/agent-rig-npm-cache agent-rig profiles --json
npx --cache /tmp/agent-rig-npm-cache agent-rig tasks create "Smoke task" --assigned-to worker --status ready
npx --cache /tmp/agent-rig-npm-cache agent-rig validate
```

## Tag

- [ ] Create release commit.
- [ ] Create tag `v0.1.1`.
- [ ] Push release branch or main.
- [ ] Push tag `v0.1.1`.

## Publish

The maintainer runs:

```bash
npm publish --access public
```

Complete npm browser authentication when prompted.

## Post-Publish Verification

```bash
npm view @inotives/agent-rig version
npx -y @inotives/agent-rig@0.1.1 --version
tmp="$(mktemp -d /tmp/agent-rig-npm-0.1.1.XXXXXX)"
cd "$tmp"
AGENT_RIG_SKIP_SKILLS=1 npx -y @inotives/agent-rig@0.1.1 init --yes
npx -y @inotives/agent-rig@0.1.1 validate
```

- [ ] npm registry reports `0.1.1`.
- [ ] npx reports `0.1.1`.
- [ ] npx scaffold and validate succeed.

## GitHub Release

After npm publish succeeds:

```bash
gh release create v0.1.1 --title "v0.1.1" --notes-file docs/release/0.1.1-notes.md
```
