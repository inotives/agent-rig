# AgentRig v0.1.3

## Highlights

- Adds an AgentRig-local `plan-tasks` skill to the built-in planner profile for phase planning and task breakdowns.
- Reframes handoff usage around planner-owned cross-session resume notes instead of routine per-task paperwork.
- Adds shared findings notes under `.agent-rig/_shared/notes/` for worker and reviewer patterns, quirks, and out-of-norm events.
- Updates `agent-rig start` to show relevant resume context pointers for recent handoffs and findings notes.

## Notes

This is a patch release for workflow hardening around planning, resume context, and durable findings. It does not change the core task lifecycle or GitHub backlog-sync surface.
