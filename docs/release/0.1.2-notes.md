# AgentRig v0.1.2

## Highlights

- Adds shared Markdown task lifecycle commands for status, assignment, type changes, blocking, unblocking, completion, and next-task selection.
- Makes `watch --once` consume the shared task workflow instead of per-agent queues.
- Adds optional GitHub Issues import through `agent-rig tasks sync github`, using the local `gh` CLI only when invoked.
- Imports GitHub Issues as unassigned `todo` backlog tasks with source metadata, label preservation, dry-run preview, and JSON output.

## Notes

This is a patch release for the task-management and GitHub backlog-sync workflow. GitHub sync is optional and requires `gh auth login` only for `agent-rig tasks sync github`.
